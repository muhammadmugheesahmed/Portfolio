A modern, responsive, and feature-rich personal portfolio built with React and Vite. This project showcases my featured work through a sleek, interactive carousel and is designed with a mobile-first approach.

**Live Demo:** [Muhammad Mughees Ahmed](https://muhammadmugheesahmed.github.io/Portfolio/)



## ✨ Features

-   **Interactive Project Carousel**: A smooth, touch-friendly carousel powered by Swiper.js to navigate through projects. It supports mouse dragging on desktop and swiping on mobile.
-   **Fully Responsive Design**: A mobile-first approach ensures a seamless and visually appealing experience on all devices, from large desktops to small smartphones.
-   **Dual-Theme Support**: A beautiful and consistent look in both **light** and **dark** modes, with UI elements adapting perfectly to the chosen theme.
-   **Dynamic Content**: Project data is managed in a centralized array, making it easy to update, add, or remove projects without touching the core component logic.
-   **Modern Tooling**: Built with **Vite** for a blazing-fast development experience and optimized build process.
-   **Custom UI Components**: Styled with pure CSS, featuring custom-designed navigation arrows and theme-aware pagination dots for a unique look and feel.

## 🛠️ Tech Stack

-   **Frontend**: React, Vite, JavaScript (ES6+)
-   **Styling**: CSS3 (with custom properties for theming)
-   **Libraries**:
    -   `Swiper.js`: For the powerful and flexible project carousel.
    -   `react-icons`: For high-quality icons.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js and npm installed on your machine.

-   Node.js (v18.x or later recommended)
-   npm

### Installation

1.  **Clone the repository:**
    ```sh
    git clone <https://github.com/muhammadmugheesahmed/react-portfolio.git>
    ```
2.  **Navigate to the project directory:**
    ```sh
    cd react-portfolio
    ```
3.  **Install NPM packages:**
    ```sh
    npm install
    ```
4.  **Run the development server:**
    ```sh
    npm run dev
    ```
    The application will be available at `http://localhost:5173` (or another port if 5173 is in use).
